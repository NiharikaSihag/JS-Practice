CSS Position Properties Explained

1. Position: Static (Default)

Elements with position: static follow the normal document flow. The top, right, bottom, and left properties have no effect.
Position: Static
(Normal flow)

2. Position: Relative

Elements with position: relative are offset relative to their normal position, but still occupy their original space in the document flow.
Position: Static
(Reference)

3. Position: Absolute

Elements with position: absolute are removed from the normal document flow and positioned relative to their nearest positioned ancestor (or the document body).

Positioned Parent (position: relative)

top: 30px, left: 30px

4. Position: Fixed

Elements with position: fixed are removed from the normal document flow and positioned relative to the viewport. They stay in place even when scrolling.
Position: Fixed
bottom: 20px, right: 20px
(Stays in corner when scrolling)

5. Position: Sticky

Elements with position: sticky behave like relative until they reach a specified scroll threshold, then they behave like fixed.

Summary of Position Properties

  • Static: Default flow, no special positioning
  • Relative: Positioned relative to its normal position, keeps its space in the flow
  • Absolute: Removed from flow, positioned relative to nearest positioned ancestor
  • Fixed: Removed from flow, positioned relative to viewport, stays in place when scrolling
  • Sticky: Hybrid of relative and fixed, becomes fixed when scroll position is reached
